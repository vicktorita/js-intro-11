1. Assignment Operator: = sign
2. Shorthand Assignment Operators: +=, -=, *=, %=, **=, /=

  let num = 10;
  num %= 4;
  console.log(num); // 2

3. Arithmetic Operators: +, -, *, /, **, ++, --

post vs pre increment/decrement 

int i = 10;

i++; // i = 10

int j = 10;

++j; // 11

4. Comparison Operators: ==, ===, !=, !==, <, >, <=, >=
*** All the comparison operators will evaluate to boolean (true or false)
== vs ===
== is used for loose comparison and it checks if only values are equal
=== is used for strict comparison and it checks if both values and data types are equal

'3' === 5   -> false 
5 == 'five' -> false 
5 == '5'    -> true
5 === '5'   -> false 


3 > 2 && 5 != 5       -> false 


&& (AND) Operator
_____ && ______ 
true  && true       -> true 
true  && false      -> false
false && true       -> false
false && false      -> false


|| (OR) Operator
_____ || ______
true  || false      -> true 
false || true       -> true
true  || true       -> true 
false || false      -> false 


! (NOT) Operator

!false    -> true 
!true     -> false 

!(5 === '5') -> true

!((3 >= 3 || false) && !true)     -> true
